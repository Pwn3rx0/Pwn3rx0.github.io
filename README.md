## pwn3rx0 | Andrew Mamdouh - Security Research Lab
This repository hosts my personal technical blog and research notes, focused on the lower-level mechanics of offensive security. My goal is to bridge the gap between "running tools" and "understanding the OS."
---
## Research Pillars
* **Malware Development & Evasion:** 34+ entries on the Windows API, focusing on custom loaders, Indirect Syscalls, and unhooking techniques. Practical implementations of IPv4 Masquerading, RC4/AES encryption, and APC Injection to bypass modern EDR/AV.
    
* **Malware Analysis:** Reverse engineering workflows using x64dbg and IDA Pro. Documentation on identifying malicious behavior, static and dynamic analysis of PE files, and decoding obfuscated strings or payloads.
    
* **Active Directory & Network Pentesting:** Comprehensive guides on attacking enterprise environments—from initial LLMNR/NBT-NS poisoning to Credential Injection and Pivoting (Lateral Movement) using tools like Impacket and BloodHound.
* **Web Application Security:** Deep dives into the OWASP Top 10, focusing on manual exploitation of SQLi, XSS, and SSRF, as well as bypassing Web Application Firewalls (WAF).
